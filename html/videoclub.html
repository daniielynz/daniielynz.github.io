Videoclub - FINALIZADA ✓

Web de Videoclub que funciona como una simulación de plataforma de gestión y compra de películas. Este es un proyecto Full Stack que tiene a Java como su lenguaje principal.

Implementamos la arquitectura Modelo-Vista-Controlador (MVC): utilizamos Servlets para la lógica de negocio en el Back-end, y JSP, JavaScript, CSS y Bootstrap para la interfaz en el Front-end y conseguir un diseño funcional. El acceso a los datos se realiza mediante clases que usan código SQL directo para gestionar la conexión con la base de datos MySQL.

#Java #Servlets #Jsp #SQL #JavaScript #CSS #Bootstrap #MVC

Características Principales

Sistema de autenticación: Gestión completa de usuarios con registro, login y perfiles personalizados.

Catálogo de películas: Navegación intuitiva con filtros avanzados, búsqueda y categorización por género.

Sistema de compra: Carrito de compra funcional con gestión de pedidos y historial de transacciones.

Gestión de inventario: Control eficiente del stock de películas mediante consultas SQL optimizadas.

Gestión de usuarios: Control de usuarios para los usuarios con rol de administradores.

Arquitectura MVC: Código organizado, escalable y fácil de mantener siguiendo las mejores prácticas.

Diseño responsive: Interfaz adaptable a cualquier dispositivo gracias a Bootstrap.
