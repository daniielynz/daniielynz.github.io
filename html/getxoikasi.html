GetxoIkasi - FINALIZADA ✓

GetxoIkasi es una aplicación móvil creada junto a los alumnos de Magisterio de la Universidad de Bilbao, diseñada para ofrecer una experiencia educativa y divertida a niños de educación infantil, integrando minijuegos interactivos y contenido didáctico.

La aplicación fue desarrollada en Android Studio. Además, incorporamos la API de Google Maps, colocando iconos en diferentes zonas del municipio de Getxo, cada uno asociado a un minijuego. Los niños debían desplazarse físicamente hasta esos puntos para desbloquearlos.
El Back-end fue implementado en Kotlin, encargándose de la lógica, la gestión de datos y la integración de los minijuegos. El Front-end se construyó utilizando XML para diseñar una interfaz atractiva, accesible y adaptada a los más pequeños. La información del progreso se maneja eficientemente mediante consultas SQL.

#Kotlin #AndroidStudio #XML #Google Maps Api #SQL #Minijuegos

Características Principales

Mapa interactivo: Integración con la API de Google Maps, mostrando iconos en distintas zonas de Getxo, cada uno asociado a un minijuego.

Minijuegos educativos: Actividades interactivas diseñadas para niños de educación infantil.

Enfoque pedagógico: Contenido desarrollado junto a alumnos de Magisterio, adaptado a la educación infantil.

Gestión de progreso: Registro eficiente del avance de cada niño mediante consultas SQL optimizadas.

Aplicación nativa: Desarrollada en Android Studio con Kotlin y XML, optimizada para usabilidad en dispositivos móviles.

Aprendizaje activo: Los niños se desplazan por el municipio para desbloquear los juegos, fomentando el Aprendizaje de sitios nuevos y la participación.
