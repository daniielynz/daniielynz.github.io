Subastas - FINALIZADA ✓

Plataforma de Subastas en línea, diseñada para la gestión de pujas y artículos. Este es un proyecto Full Stack que tiene a PHP como su lenguaje principal.
Implementé la arquitectura Modelo-Vista-Controlador (MVC): el Back-end se basa en PHP, utilizando código SQL directo para gestionar las consultas y transacciones en la base de datos MySQL. El Front-end se construyó con HTML y CSS.

Como elemento innovador, he integrado un chatbot de asistencia utilizando JavaScript, la plataforma de automatización n8n y el modelo de lenguaje Llama3, añadiendo una capa de inteligencia artificial al servicio.

#PHP #SQL #HTML #JavaScript #CSS #MVC #LLama3 #n8n

Características Principales

Sistema de autenticación: Gestión completa de usuarios con registro, login y perfiles personalizados.

Listado de productos en venta: Navegación intuitiva con filtros avanzados, categorización por tipo de producto.

Sistema de puja: Opción de puja funcional solo para los usuarios autenticados (como se muestra en las imágenes)

Gestión de inventario: Control eficiente del stock de productos mediante consultas SQL optimizadas.

Arquitectura MVC: Código organizado, escalable y fácil de mantener siguiendo las mejores prácticas.

Diseño responsive: Interfaz adaptable a cualquier dispositivo gracias a Bootstrap.

Chatbot de ayuda: Asistencia automática a los usuarios mediante respuestas inteligentes.
